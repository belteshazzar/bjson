Quick Start

Node.js

const { encode, decode, ObjectId } = require('./bjson.js'); // Encode data to binary const data = { name: 'John', age: 30 }; const binary = encode(data); // Decode binary back to data const decoded = decode(binary); console.log(decoded); // { name: 'John', age: 30 }

Browser (OPFS)

const { BJsonFile } = window.BJson; // Write to file const file = new BJsonFile('data.bjson'); await file.write({ name: 'John', age: 30 }); // Read from file const data = await file.read(); console.log(data); // { name: 'John', age: 30 }

Type Encoding

Type Byte Format
NULL 0x00 No additional data
FALSE 0x01 No additional data
TRUE 0x02 No additional data
INT 0x03 4 bytes (32-bit signed, little-endian)
FLOAT 0x04 8 bytes (64-bit float, little-endian)
STRING 0x05 4-byte length + UTF-8 bytes
OID 0x06 12 bytes (MongoDB ObjectId)
ARRAY 0x10 4-byte length + encoded elements
OBJECT 0x11 4-byte count + key-value pairs
